Dulcero de Lata

es un recipiente decorativo que se utiliza para guardar dulces y otros tipos de golosinas. Los dulceros de lata suelen estar hechos de metal, como aluminio o acero, y suelen tener una tapa hermética para mantener los dulces frescos.

Dulcero de lata

Materiales necesarios:

Latas de aluminio

Pintura en aerosol de diferentes colores

Pincel

Tijeras

Pegamento

RAMO DE ROSAS CON PEDIORICO

es una forma creativa y original de reciclar papel. El papel periódico se puede usar para crear pétalos de rosa de diferentes tamaños y colores. Estos pétalos se pueden unir para crear un ramo de rosas único y llamativo.

Rosas de Pediorico
